Tablo Yapılması

Tablolar table etiketi içerisine yazılır,
tr--> Table Row ile çevrelenir.
th --> table header etiketine başlıklar
td--> Table data tagıne içerikler yazılır.
boreder --> attiributi ile tabloya çizgi/kalınlık verileri.
colspan --> yatay uzat
rowspan --> dikey uzat
allign--> attirubite ile default olarak ortalanan başlıklar ile sola yaslı olan satır yazıları değiştirilir.
thead--> içine th ile başılar yazılıyor. ortalanmış/kalın olarak yazılır.
tbody --> içine td'ler yani içerikler yazılır.
